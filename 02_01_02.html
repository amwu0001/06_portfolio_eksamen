02.01.02 Website

I tema 2 blev vi introduceret til grundlæggende redskaber og begreber indenfor multimediedesign. Denne viden anvendte jeg i opgave 02.01.02, hvor jeg opbyggede et responsivt website ved brug af HTML og CSS. Ved at følge det udleverede wireframe og layoutdiagram, lærte jeg at anvende CSS grid, flex og media quieries.

I min løsning tog jeg højde for designkonventioner og gestalltprincipper. Dette gjorde jeg ved bl.a. at gøre brug af CSS declarations som margin, font-weight og color for at skabe overblik, kontinuitet og hierarki. Jeg stødte på enkelte problemer indenfor billedebehandling og CSS grid på galleri.html og min_side.html. Jeg gjorde derfor brug af udviklingsværktøjet inspector i Firefox, og fandt frem til at jeg kunne løse problemet ved at ændre på margin/padding i CSS og billedebeskæring i Photoshop.
